# Flask Student Portal v1

A complete mini-web application demonstrating advanced Python OOP, modular project structuring, and a fully functional Flask web portal with HTML templates.

##  Features (Tasks 1-5)
- **Advanced OOP:** Custom magic methods, `@property`, and class/static methods.
- **Modular Architecture:** Organized packages (`stu_co`), isolated environments (`venv`), and clean code (`flake8`).
- **Flask Web Server:** Handles `GET` & `POST` requests with a dynamic routing system.
- **Jinja2 Templates:** Dynamic HTML pages with custom CSS styling and a responsive Navbar.
- **Full Portal System:** Register students, view the student list, and check individual GPA/grades.
- **Testing:** Automated unit tests for OOP logic and Flask routes using `pytest`.
